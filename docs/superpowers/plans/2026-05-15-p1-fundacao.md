# P1 — Fundação (Wenox PWA CRM) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Entregar um app Ionic React PWA instalável que autentica um usuário da Wenox contra o PocketBase e exibe um shell autenticado vazio, com pipeline de deploy ativo em `app.wenox.com.br`.

**Architecture:** SPA Ionic React (Vite + TS) servida como build estático pelo EasyPanel. Autenticação via coleção `usuarios` (auth) do PocketBase usando o SDK oficial; sessão persistida em `localStorage`. Rotas protegidas via um `AuthContext`. PWA via `vite-plugin-pwa`.

**Tech Stack:** Ionic React 8, Vite 5, TypeScript, Capacitor 6, `pocketbase` JS SDK, `vite-plugin-pwa`, Vitest + @testing-library/react.

**Backend já provisionado:** PocketBase em `https://api.wenox.com.br` (superadmin `adm@wenox.com.br`). Repo: `github.com/DigitalBoostofc/wenox-pwa-crm`. Subdomínio `app.wenox.com.br` já aponta para a VPS (sem serviço ainda).

- [ ] **Step 3: Implementar `src/auth/AuthContext.tsx`**

```tsx
import { createContext, useEffect, useMemo, useState, ReactNode } from 'react';
import { pb } from '@/lib/pocketbase';

export interface AuthUser {
  id: string;
  email: string;
  role: string;
  nome?: string;
}

export interface AuthContextValue {
  user: AuthUser | null;
  login: (identity: string, password: string) => Promise<void>;
  logout: () => void;
}

export const AuthContext = createContext<AuthContextValue | null>(null);

export function AuthProvider({ children }: { children: ReactNode }) {
  const [user, setUser] = useState<AuthUser | null>(
    pb.authStore.isValid ? (pb.authStore.record as unknown as AuthUser) : null
  );

  useEffect(() => {
    const unsub = pb.authStore.onChange(() => {
      setUser(
        pb.authStore.isValid ? (pb.authStore.record as unknown as AuthUser) : null
      );
    });
    return () => unsub();
  }, []);

  const value = useMemo<AuthContextValue>(
    () => ({
      user,
      login: async (identity, password) => {
        await pb.collection('usuarios').authWithPassword(identity, password);
      },
      logout: () => pb.authStore.clear(),
    }),
    [user]
  );

  return <AuthContext.Provider value={value}>{children}</AuthContext.Provider>;
}
```

```tsx
import { useState } from 'react';
import {
  IonContent, IonInput, IonButton, IonPage, IonText, IonItem, IonList,
} from '@ionic/react';
import { useAuth } from '@/auth/useAuth';

export function LoginPage() {
  const { login } = useAuth();
  const [email, setEmail] = useState('');
  const [senha, setSenha] = useState('');
  const [erro, setErro] = useState('');
  const [carregando, setCarregando] = useState(false);

  async function onSubmit(e: React.FormEvent) {
    e.preventDefault();
    setErro('');
    setCarregando(true);
    try {
      await login(email, senha);
    } catch {
      setErro('Credenciais inválidas');
    } finally {
      setCarregando(false);
    }
  }

  return (
    <IonPage>
      <IonContent className="ion-padding">
        <h1 style={{ textAlign: 'center', marginTop: 48 }}>Wenox ⚡</h1>
        <form onSubmit={onSubmit}>
          <IonList>
            <IonItem>
              <IonInput
                label="Email"
                labelPlacement="floating"
                type="email"
                value={email}
                onIonInput={(e) => setEmail(e.detail.value ?? '')}
                aria-label="Email"
              />
            </IonItem>
            <IonItem>
              <IonInput
                label="Senha"
                labelPlacement="floating"
                type="password"
                value={senha}
                onIonInput={(e) => setSenha(e.detail.value ?? '')}
                aria-label="Senha"
              />
            </IonItem>
          </IonList>
          {erro && (
            <IonText color="danger">
              <p>{erro}</p>
            </IonText>
          )}
          <IonButton expand="block" type="submit" disabled={carregando}>
            Entrar
          </IonButton>
        </form>
      </IonContent>
    </IonPage>
  );
}
```
